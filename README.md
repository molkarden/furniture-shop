# 🪑 Уютный дом — Интернет-магазин мебели

Полноценный интернет-магазин мебели на Django с каталогом, корзиной, оформлением заказов и личным кабинетом.

## 🛋 Возможности

- 🛍 Каталог мебели с карточками товаров
- 📄 Страница товара с характеристиками, описанием и ценой
- 🛒 Корзина с добавлением/удалением товаров
- 📦 Оформление заказа (доставка курьером или самовывоз)
- 📋 Личный кабинет с историей заказов
- 🔐 Регистрация/вход (email + пароль)
- 🔑 Восстановление пароля
- 📞 Страница контактов с формой обратной связи
- 📱 Адаптивный премиальный дизайн

## 🎨 Дизайн

- **Цвета:** золотой (#c9a96e) + тёмный (#1a1a2e)
- **Шрифты:** Playfair Display (заголовки) + Inter (текст)
- Плавные анимации, тени, hover-эффекты

## 🛠 Технологии

- **Backend:** Python 3, Django 4
- **База данных:** SQLite (по умолчанию) или PostgreSQL
- **Frontend:** HTML5, CSS3 (Grid, Flexbox)
- **Аутентификация:** Django Sessions
